##  Стек технологий

*   **Язык программирования:** C#
*   **Платформа:** .NET (версия, например, .NET 6.0 / .NET Framework 4.7.2)
*   **Технология GUI:** Windows Presentation Foundation (WPF)
*   **Элементы управления:**
    *   `Page` - для размещения интерфейса.
    *   `TextBox` - для ввода данных (x0, xk, dx).
    *   `Button` - для запуска вычислений и очистки.
    *   `Canvas` - для рисования графика функции.
    *   `TextBlock`/`Label` - для подписей.
*   **Система контроля версий:** Git (платформа GitVerse.ru)
